# Wordnnect

## これは何？
このゲームは、2022年12月に行われたバンダイナムコスタジオの冬季インターンの成果物です。
多言語に対応したバイナリデータを使用したゲームという条件の元、言葉遊びをテーマにしたゲームを思いついたので、制作しました。

## 遊び方
『同じ意味を持つ単語同士をつなげる』

たったこれだけです。
今まで繋いだ数と、どれだけ早く繋げられるかで得点が左右されます。

ゲームプレイ中にQを押すとタイトルに戻ります。
F1を押すと、ライセンス情報を表示できます。
